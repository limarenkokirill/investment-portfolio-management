# Investment Portfolio Management

A modern investment portfolio management application built with **SAP ABAP Cloud** using the **RESTful Application Programming Model (RAP)** on **SAP BTP**.

The project demonstrates modern SAP development practices including RAP Business Objects, CDS View Entities, OData V4 services, EML, Behavior Definitions, Clean Core principles, and Git-based development workflows.

## Features

* Portfolio management
* Investment transactions (Buy / Sell)
* Stocks, Bonds and ETFs
* Portfolio valuation
* Profit & Loss calculation
* Market price management
* Fiori Elements UI
* RESTful OData V4 API

## Technology Stack

* SAP ABAP Cloud
* SAP BTP ABAP Environment
* ABAP Development Tools (ADT)
* RAP (RESTful Application Programming Model)
* CDS View Entities
* OData V4
* Git & GitHub

## Project Goals

This project has two main objectives:

1. Prepare for the **SAP Certified Backend Developer – ABAP Cloud** practical certification.
2. Showcase modern SAP development skills through a realistic enterprise application.

## Engineering Principles

This project is built around the following engineering principles:

- Design before implementation
- Business drives technology
- Clean Core over quick fixes
- Readability over cleverness
- Simplicity over complexity
- Documentation is part of the product
- Git history tells the story of the project

The application follows Clean Core principles and is developed using cloud-ready SAP technologies and best practices.
